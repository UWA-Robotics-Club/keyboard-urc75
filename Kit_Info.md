- 1x Resin 3D Printed Top

- Solder 
- M2 Allen key / driver
- Masking Tape

- **USB C Cable**

- USB C Cable **THIS KIT DOESN NOT INCLUDE A USB C CABLE**
- Soldering iron and solder
- Phillips Screwdriver 
- M2 Allen Key
- Side Cutters
- (Helpful Option) Masking Tape
